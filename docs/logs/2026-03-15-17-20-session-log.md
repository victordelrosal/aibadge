## Gotchas & Lessons Learned
- **file:// protocol + CDN**: Three.js CDN loads fine from file://, but the texture loader can have timing issues; testing via localhost is more reliable
- **CylinderGeometry UV mapping**: Cap UVs don't account for post-construction `rotateX/Z` transforms; requires `texture.center.set(0.5,0.5)` + `texture.rotation` to fix orientation
- **GitHub Pages CDN cache**: Deployment builds succeed immediately but CDN propagation can lag several minutes; curl with cache-bust confirms deployment before browser shows it
- **Container display:none breaks sizing**: Measuring `clientWidth` on a hidden container returns 0; need to temporarily show with `visibility:hidden` or use a fallback default size
- **Commit history as safety net**: Keeping clean commit points (8ccfc61, d44550b) before major changes enabled quick reverts when experiments (glossy streak) didn't land
