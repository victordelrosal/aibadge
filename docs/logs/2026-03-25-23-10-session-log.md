## Gotchas & Lessons Learned
- Firebase Blaze plan has a max linked project quota per billing account. No workaround except creating a new billing account or unlinking other projects
- Resend requires domain verification per account. The victordelrosal account only had sandbox access for fiveinnolabs.com domain
- Email HTML must use table-based layouts. `display:flex`, `border-radius` on score circles, and CSS gradients are unreliable across email clients
- `git add -A` will capture node_modules if no .gitignore exists. Always add .gitignore first
- The score circle in email uses a `border: 6px solid #D4AF37` trick on a table cell with fixed width/height and border-radius
