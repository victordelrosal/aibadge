Detailed Step-by-Step Guide

Complete instructions with screenshots

Part 1: Generate Your Code in a Large Language Model

Step 1: Open an LLM

Go to chat.openai.com, gemini.google.com, or claude.ai

Step 2: Enter the Prompt

Copy and paste this exact text:

Code my very first Hello World page in under 50 lines of code that includes a button and some movement. Output as a single-file HTML, JavaScript, CSS.

Step 3: Copy the Code

The LLM will generate code in a grey box. Copy it using:

  • Click the "Copy code" button (top-right of code block), OR
  • Click inside code, then Cmd+A / Ctrl+A (select all), then Cmd+C / Ctrl+C (copy)

Important

Copy ALL the code: it must start with <!DOCTYPE html> and end with </html>

Part 2: Install Sublime Text

Mac Installation

  • Go to sublimetext.com and click "Download for Mac"
  • Open the downloaded .dmg file from your Downloads folder
  • Drag the Sublime Text icon into the Applications folder
  • Open from Applications (if security warning appears: System Preferences > Security > Open Anyway)

Windows Installation

  • Go to sublimetext.com and click "Download for Windows"
  • Run the downloaded .exe file
  • Click Next > Next > Install > Finish

Part 3: Create and Save Your HTML File

Step 1: Paste Your Code

Open Sublime Text > Click inside the window > Paste (Cmd+V / Ctrl+V)

You should see colorful code.

Step 2: Save As HTML

  • Go to File > Save As (or Cmd+Shift+S / Ctrl+Shift+S)
  • Navigate to your Desktop (so you can find it easily)
  • Type the filename: helloworld.html
  • Click Save

Critical

The .html extension is essential! Without it, browsers won't recognize your file as a webpage.

Filename Examples

Wrong
helloworldhelloworld.txt
Correct
helloworld.htmlmypage.html

Part 4: Open in Chrome

Choose any method:

  • Double-click the helloworld.html file on your Desktop
  • Drag & drop the file into an open Chrome window
  • Right-click > Open with > Google Chrome

Congratulations!

You've created your first webpage using HTML, CSS, and JavaScript!

Troubleshooting

Common problems and solutions

Pro Tip: Use AI as Your IT Support

Stuck on an error? Paste the error message or screenshot directly into ChatGPT, Claude, or Gemini. These AI tools can read error messages and guide you step-by-step to fix problems.

Common Issues

"File opens as text, not a webpage"

File wasn't saved with .html extension. Rename the file or save again with the correct extension.

"Can't find my file"

Search for "helloworld" in Finder (Mac) or File Explorer (Windows).

"Page is blank or shows weird characters"

Re-copy the code from the LLM and paste into a new file. Make sure you copied ALL the code.

"Sublime Text won't open on Mac"

Go to System Preferences > Security & Privacy > Click "Open Anyway"

Common Mistakes to Avoid

  • Saving as helloworld.txt instead of helloworld.html
  • Not copying ALL the code (must start with <!DOCTYPE html>)
  • File opens as text? Rename it to end in .html
