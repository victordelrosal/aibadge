# Inkwell Books

Inkwell is an independent bookshop chain. Footfall is down and the loyalty scheme is a paper stamp card. There is no email list.

Produce a re-engagement plan with one working artifact. Use Octopus to run the pipeline.

You should:
- Diagnose why younger customers left and size the opportunity.
- Design a re-engagement concept.
- Build a real sign-up landing page.
- Draft the launch email and three social posts.
