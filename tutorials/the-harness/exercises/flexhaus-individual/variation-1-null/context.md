# FlexHaus Gym

FlexHaus is a single-location boutique gym. Member churn is high and class attendance is patchy. Many members never come back after their first class.

Produce a member-retention plan with one working artifact. Use Octopus to run the pipeline.

You should:
- Diagnose the churn and find the at-risk segment.
- Design a retention concept.
- Build a real booking or sign-up page with sales copy to win members back.
